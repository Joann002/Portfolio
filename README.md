# Portfolio — Joann Michel Razafinimanana

Portfolio personnel d'un développeur Fullstack — site statique (HTML / CSS / JavaScript), sans dépendance externe au runtime.

🌐 **En ligne :** _(lien GitHub Pages à venir)_

## ✨ Points clés

- Design « constellations » avec glassmorphism et interactions animées
- Bilingue **FR / EN** (i18n côté client)
- 100 % statique : aucune build, aucune dépendance à installer
- Icônes technos en SVG inline (aucun appel réseau)
- Responsive et respectueux de `prefers-reduced-motion`

## 🗂️ Structure

```
index.html        # Page unique
css/styles.css    # Styles + tokens de design
js/main.js        # i18n, animations, interactions
assets/           # Images et CV
```

## 🚀 Lancer en local

Ouvrir `index.html` dans un navigateur, ou servir le dossier :

```bash
python3 -m http.server 8000
# puis http://localhost:8000
```
